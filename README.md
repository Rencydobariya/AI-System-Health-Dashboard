# 🚀 AI System Health Dashboard

## 📌 Overview
This project is a smart system monitoring dashboard that shows real-time CPU, RAM, and Disk usage with an attractive UI. It also analyzes system risk and provides suggestions to improve performance.

---

## ✨ Features
- 📊 Live System Monitoring (CPU, RAM, Disk)
- 🔥 Interactive Pie Chart & Bar Graph
- ⚠️ Risk Detection System (Safe / Medium / High)
- 💡 Smart Suggestions based on system usage
- 🤖 Simple AI Chatbot for software-related queries
- 🎨 Attractive UI with animations and colors

---

## 🛠 Technologies Used
- Python
- Streamlit
- Plotly
- Psutil
